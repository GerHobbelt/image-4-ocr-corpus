The file DIBCO2016_dataset-original.zip contains the testing dataset which comprises 10 handwritten
images.

The file DIBCO2016_dataset-GT.zip contains the associated ground truth.

The file BinEvalWeights.zip contains the executable ‘BinEvalWeights.exe’ that produces the weights which are required to run the evaluation program.

The file DIBCO_metrics.zip contains the evaluation program. It needs the MCR 9.0 from https://www.mathworks.com/products/compiler/mcr.html
